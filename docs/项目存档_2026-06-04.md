## 自研独创性亮点

| 模块 | 独创内容 |
|------|---------|
| 毛利率公式 | `(营收 - 营业成本 - 税金及附加) / 营收`（中国准则版，IFRS 版不含税金及附加） |
| 速动比率 | `(流动资产 - 存货) / 流动负债`（中国中小企业无"有价证券"科目） |
| 利息保障倍数 | `(利润总额 + abs(财务费用)) / abs(财务费用)`（中小企业不单独列利息支出） |
| EBITDA | 三段式替代估算：`EBIT → +折旧摊销 → +资产减值 → 兜底` |
| 实际税率 | 含利润总额为负时返回 NaN 的边界保护 |
| Altman Z-score | X5 系数 0.999（中国学者修正参数，非标准 1.0） |
| 科目映射 | 80+ 手写条目 + difflib 模糊匹配兜底 |

## 已知限制

1. **PDF 不支持**: 电子/扫描 PDF 均已取消，仅 Excel
2. **成长指标**: 需多期数据，单期标注 N/A
3. **行业基准**: 未接入 AKShare，基准值使用通用阈值
4. **EBITDA**: 采用三段式替代估算
5. **Altman Z-score**: 使用中国学者修正参数 (默认张玲参数), 可配置
6. **著作权**: 代码 ~70% 自研（中国准则适配层完全独立），6 个 source code/ 开源项目均未引用
